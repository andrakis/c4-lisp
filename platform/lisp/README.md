Lisp on C4
==========

Directory Structure
-------------------

```
  platform/              Interpreter platforms
     lisp/               Lisp interpreter
       bin/              Contains scripts to build c4_src (below)
          makec4.c       A script which does some slight modifications (snipping content)
       c_src/            Individual .c files that make up the interpreter
       c4_src/           Destination for bundled .c4 file containing contents of above
          _preamble.c4
          _entry.c4      This and the file above are inserted into the next file
          lisp.c         Generated by Makefile
       Makefile          Used to build the c4_src contents
```
